🌍 Earth Observation Pipeline for Land-Use ClassificationThis repository implements a geospatial machine learning pipeline for land-use classification using Sentinel-2 satellite imagery and ESA WorldCover 2021 raster data.The project bridges the gap between geospatial analysis (handling coordinate systems and rasters) and deep learning (training a CNN for computer vision tasks).📁 Repository StructurePlaintext.

🛰️ Dataset Overview1. Sentinel-2 RGB Image PatchesDimensions: 128 × 128 pixels.Resolution: 10 meters per pixel.Format: PNG files where the filename encodes the center coordinates (e.g., 28.2056_76.8558.png).2. ESA WorldCover 2021Source: Global land-cover dataset at 10m resolution.Function: Serves as the ground truth provider. The pipeline computes the statistical mode of the classes within a 128x128 window to assign a single label to each satellite patch.3. Study Area: Delhi NCRBoundary: Defined via a GeoJSON/Shapefile.Projection: Data is reprojected from EPSG:4326 (WGS84) to EPSG:32644 (UTM Zone 44N) to allow for distance-based spatial reasoning in meters.⚙️ Pipeline WorkflowThe pipeline is divided into three distinct modular stages:1️⃣ Spatial Reasoning & FilteringCRS Transformation: Converting global coordinates to local UTM meters.Spatial Join: Using GeoPandas to filter 9,216 candidate images down to 8,015 images that strictly fall within the Delhi NCR boundary.Grid Generation: Creating a 60 × 60 km hexagonal or square grid using Shapely for regional visualization.2️⃣ Label ConstructionThe pipeline performs a "Raster-to-Image" lookup:Map the image center to the specific pixel in the ESA WorldCover TIFF.Extract the surrounding 128 × 128 pixel window.Map the ESA numeric codes to human-readable classes:ESA CodeLabelESA CodeLabel10Tree Cover50Built-up20Shrubland60Bare/Sparse Veg30Grassland80Perm. Water40Cropland90Herbaceous Wetland3️⃣ Deep Learning (ResNet-18)A ResNet-18 architecture is fine-tuned to classify the RGB patches.ParameterValueArchitectureResNet-18 (Pre-trained)OptimizerAdamLoss FunctionCross-Entropy LossAugmentationFlips, Rotation, Gaussian Noise (Albumentations)Split Ratio60% Train / 40% Test (Stratified)📈 Results & EvaluationThe model was evaluated using standard classification metrics to account for the class imbalance inherent in urban-rural fringe datasets.Test Accuracy: ~0.66Weighted F1 Score: ~0.61[!IMPORTANT]Observation: The model shows high recall for Cropland and Built-up classes. This reflects the reality of the Delhi NCR region, where these two classes dominate the landscape. Future iterations could use oversampling or Focal Loss to improve minority class (e.g., Water, Wetland) detection.🔁 Reproducibility1.
